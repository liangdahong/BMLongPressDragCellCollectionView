## Introduction

- 本框架是一个让你轻松实现类似支付宝的拖拽重排功能，支持自定义，iOS8+，具体可查看代码。

##  CocoaPods

```ruby
pod 'BMLongPressDragCellCollectionView'
```

```ruby
pod install
```

```objective-c
#import "BMLongPressDragCellCollectionView.h"
```
